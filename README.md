# Angular

## Prerrequisites 

- [Node js](https://nodejs.org/) 
- Download Angular:

  ```BASH
  npm install -g @angular/cli@latest
  ```
  
- Verify Installation:
  ```BASH
  ng version
  ```

# Create New Project
  
  ```BASH
  ng new project_name
  ```

# Create Component

  ```BASH
  ng g c component_name
  ```

# Executing project

  ```BASH
  ng serve -o
  ```
